<p>Der <b>Sachsenspiegel</b> (<a href="https://de.m.wikipedia.org/wiki/Nieders%C3%A4chsische_Sprache" class="mw-redirect" title="Niedersächsische Sprache">nds.</a> <i>Sassenspegel</i>, <a href="https://de.m.wikipedia.org/wiki/Mittelniederdeutsche_Sprache" title="Mittelniederdeutsche Sprache">mittelniederdeutsch</a>: <i>Sassen Speyghel</i>) ist ein <a href="https://de.m.wikipedia.org/wiki/Rechtsbuch" title="Rechtsbuch">Rechtsbuch</a> des <a href="https://de.m.wikipedia.org/wiki/Eike_von_Repgow" title="Eike von Repgow">Eike von Repgow</a>, entstanden zwischen 1220 und 1235. Es gilt als das bedeutendste und, gemeinsam mit dem <a href="https://de.m.wikipedia.org/wiki/M%C3%BChlh%C3%A4user_Reichsrechtsbuch" title="Mühlhäuser Reichsrechtsbuch">Mühlhäuser Reichsrechtsbuch</a>, älteste Rechtsbuch des deutschen <a href="https://de.m.wikipedia.org/wiki/Hochmittelalter" title="Hochmittelalter">Mittelalters</a>. Zugleich ist der <i>Sachsenspiegel</i> die erste in <a href="https://de.m.wikipedia.org/wiki/Mittelniederdeutsch" class="mw-redirect" title="Mittelniederdeutsch">mittelniederdeutscher</a> Sprache verfasste <a href="https://de.m.wikipedia.org/wiki/Prosa" title="Prosa">Prosaliteratur</a>.</p>
<p>Die Bezeichnung <i>Sachsenspiegel</i> beruht auf der Zugehörigkeit zur <a href="https://de.m.wikipedia.org/wiki/Spiegel_%28Literatur%29" title="Spiegel (Literatur)">Spiegelliteratur</a>
 der Entstehungszeit. Er war vorrangig deutsch-rechtlich, enthielt aber 
auch wenige Teile römischen und kanonischen Rechts; die neuere Forschung
 hebt in diesem Zusammenhang die kirchenrechtlichen Einflüsse hervor. <a href="https://de.m.wikipedia.org/wiki/Lied" title="Lied">Lieder</a> und <a href="https://de.m.wikipedia.org/wiki/Epos" title="Epos">Epen</a> wurden ebenfalls in <a href="https://de.m.wikipedia.org/wiki/Dialekt" title="Dialekt">Dialekten</a> geschrieben.</p>

</div></div><h2 aria-controls="content-collapsible-block-0" aria-haspopup="true" tabindex="0" class="in-block section-heading collapsible-heading"><div class="mw-ui-icon mw-ui-icon-arrow mw-ui-icon-element  indicator" title=""></div><span class="mw-headline" id="Historische_Hintergr.C3.BCnde">Historische Hintergründe</span><span><a href="https://de.m.wikipedia.org/w/index.php?title=Sachsenspiegel&amp;action=edit&amp;section=1" title="Abschnitt bearbeiten: Historische Hintergründe" data-section="1" class="mw-ui-icon mw-ui-icon-element mw-ui-icon-edit-enabled edit-page">Bearbeiten</a></span></h2><div aria-expanded="false" aria-pressed="false" class="mf-section-1 collapsible-block" id="content-collapsible-block-0">
<p>Jede <a href="https://de.m.wikipedia.org/wiki/Rechtsgeschichte" title="Rechtsgeschichte">rechtshistorische</a> Epoche ist durch die Art ihrer <a href="https://de.m.wikipedia.org/wiki/Rechtsquelle" title="Rechtsquelle">Rechtsquellen</a> gekennzeichnet. Im <a href="https://de.m.wikipedia.org/wiki/Hochmittelalter" title="Hochmittelalter">Hochmittelalter</a>, das bis ins 13. Jahrhundert reichte, zersplitterte das <a href="https://de.m.wikipedia.org/wiki/Germanische_Stammesrechte" title="Germanische Stammesrechte">Recht der einzelnen germanischen Stämme</a>. Eine Fülle von <a href="https://de.m.wikipedia.org/wiki/Landesrecht" title="Landesrecht">Landes</a>- und <a href="https://de.m.wikipedia.org/wiki/Ortsrecht" title="Ortsrecht">Ortsrechten</a> entstand, darunter als besondere Gruppe die <a href="https://de.m.wikipedia.org/wiki/Landfrieden" title="Landfrieden">Landfrieden</a>. Im folgenden <a href="https://de.m.wikipedia.org/wiki/Sp%C3%A4tmittelalter" title="Spätmittelalter">Spätmittelalter</a> gab es neben dem umfangreich <a href="https://de.m.wikipedia.org/wiki/Rezeption_des_r%C3%B6mischen_Rechts" title="Rezeption des römischen Rechts">rezipierten römischen Recht</a> regionale Quellengruppen: Rechtsbücher, <a href="https://de.m.wikipedia.org/wiki/Stadtrecht" title="Stadtrecht">Stadtrechte</a> und ländliche <a href="https://de.m.wikipedia.org/wiki/Weistum" title="Weistum">Weistümer</a>.</p>
<p>Mittelalterliches Recht war mündlich überliefertes Gewohnheitsrecht. 
Es zeichnete sich aus durch Alter, Bewährung und Verständlichkeit. 
Systematische Geschlossenheit, begriffliche Klarheit und logische <a href="https://de.m.wikipedia.org/wiki/Stringenz" class="mw-redirect" title="Stringenz">Stringenz</a> waren nicht oder nur ansatzweise vorhanden.<sup id="cite_ref-WDL_1-0" class="reference"><a href="#cite_note-WDL-1">[1]</a></sup></p>
